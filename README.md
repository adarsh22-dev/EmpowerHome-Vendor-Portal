# Nexus Marketplace - Full-Stack E-Commerce Platform

Nexus is a comprehensive, multi-vendor e-commerce marketplace built with a modern tech stack. It supports retail, wholesale, and a unique product exchange system.

## 🚀 Core Features

### 🛍️ Shopping Experience
- **Advanced Product Catalog**: Categories, detailed product specs, and multi-image galleries (including 360° views).
- **Smart Search & Filtering**: Find products by category, price, and vendor.
- **Wishlist & Cart**: Persistent shopping cart and wishlist for registered users.
- **Product Reviews**: Verified customer ratings and comments with admin moderation.
- **Product Videos**: Integrated video support for product demonstrations.
- **Subscriptions**: Weekly, monthly, or quarterly subscription options for recurring purchases.

### 🏢 Multi-Vendor System
- **Vendor Portal**: Dedicated settings for store branding, policies (shipping/return), and SEO.
- **Staff Management**: Vendors can create staff accounts with specific permissions.
- **Shipping Methods**: Vendor-specific shipping rates and estimated delivery times.
- **Analytics**: Vendor-specific sales tracking and performance metrics.

### 📦 Wholesale & B2B
- **Wholesale Pricing**: Tiered pricing based on quantity for bulk buyers.
- **Wholesaler Roles**: Special user roles with access to exclusive wholesale rates.
- **Bulk Enquiries**: Direct communication for sample requests and bulk orders.

### 🔄 Unique Exchange System
- **Product Exchange**: Customers can request to exchange old products for credit towards new purchases.
- **IMEI Validation**: Secure validation for electronic device exchanges.
- **Exchange Management**: Admin interface to approve/reject exchange requests and set valuation rules.

### 💰 Marketing & Loyalty
- **Loyalty Program**: Earn points on every purchase; track balances and history.
- **Coupon System**: Percentage or fixed-amount discounts with usage limits and expiry dates.
- **Banner Management**: Dynamic homepage hero banners managed via the admin dashboard.
- **Promotions**: Vendor-specific discount codes and flash sales.
- **Smart Recommendations**: Upsells and cross-sells on product, cart, and checkout pages.

### 🛠️ Admin Command Center
- **Global Stats**: Real-time revenue, order, and user analytics.
- **User Management**: Full control over customers, vendors, and wholesalers.
- **CMS Builder**: Customize homepage sections, blog posts, and FAQ.
- **Support Center**: Integrated ticket system for customer and vendor support.
- **Platform Settings**: Configure site-wide branding, tax classes, and system-wide toggles.

## 💻 Tech Stack

- **Frontend**: React 18, Next.js (App Router), Tailwind CSS v4.
- **Animations**: Framer Motion (`motion/react`).
- **Icons**: Lucide React.
- **Charts**: Recharts for data visualization.
- **Backend**: Node.js, Express 5.
- **Database**: SQLite (`better-sqlite3`) for high-performance local storage.
- **Type Safety**: TypeScript across the entire stack.

## 🛠️ Getting Started

1. **Environment**: Ensure Node.js is installed.
2. **Installation**: `npm install`
3. **Development**: `npm run dev`
4. **Build**: `npm run build`
5. **Production**: `npm start`
